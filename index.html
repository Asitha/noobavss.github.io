NoobaVSS Video Surveillance System

Deploy your own Smart Surveillance System now!

NoobaVSS is free and open source smart surveillance system.It is based on Qt and use OpenCV for image processing.

NoobaVSS

NoobaVSS is the main platform which serves plug-ins, and provide user interface.

NoobaVSS Plug-ins

Plug-ins consist of different functionalities suitable for different tasks. As a collection of plug-ins, any specific surveillance context can be addressed.

Several widely applicable set of plug-ins are available. You can build your own too.
